# Portal Conexão Segura 🌐
> Projeto de conscientização sobre Cidadania Digital e os impactos da Inteligência Artificial.

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como atividade de Recuperação da disciplina de Educação Digital e IA (Ensino Médio). O objetivo principal é atuar como um portal comunitário focado em educar os cidadãos sobre os perigos da desinformação automatizada e o surgimento das mídias manipuladas por IA (Deepfakes). 

O portal traz notícias educativas, um sistema dinâmico de triagem de veracidade de links e um horóscopo do dia adaptado com conselhos éticos digitais.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da página (`main`, `section`, `article`, `form`).
* **CSS3:** Estilização com Layout Avançado Flexbox, Design Responsivo (Media Queries) e gerenciamento de temas via Variáveis CSS.
* **JavaScript (Vanilla):** Manipulação dinâmica do DOM, tratamento de eventos do usuário, lógica condicional para filtragem de palavras-chave e controle de acessibilidade (Modo Escuro).

## 🚀 Como Executar o Projeto
1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente em qualquer navegador moderno.
3. O projeto também está publicado oficialmente via **GitHub Pages**.
